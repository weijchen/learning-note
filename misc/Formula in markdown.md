# 方法一：MathJax 引擎
    * 在網頁上顯示主要使用到 MathJax，是一个 JavaScript 庫，可以跨瀏覽器渲染數學公式的引擎，其語法類似 LaTeX，支持 LaTeX, MathML 和 AsciiMath 標記，Open-Source
    * Official website: https://www.mathjax.org/
    * 代碼使用有兩種方式：
        * 行間公式（獨立出一行的方程式）：double $$
            `$$x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}$$`
        * 行內公式（可與前後文並行）：singal $
            `$x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}$`
